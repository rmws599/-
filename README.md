# 手持小风扇 PCB 资料

## 项目概述
这是一个基于FM5012f芯片设计的手持小风扇的完整硬件设计资料，包含原理图、PCB 源文件及实物验证照片。

## 📁 目录结构说明

- **设计源文件**：
  - `手持风扇_2026-07-08.epro2`：嘉立创 EDA 源文件。
  - `手持小风扇1.0.zip`：Altium Designer (AD) 工程包。

- **预览图片**：
  - `原理图.png`：电路原理图预览。
  - `PCB1.png`：PCB 版图预览。
  - `实物1.jpg`, `实物2.jpg`：组装后的实物运行图（含电机与电池）。

## 使用说明
下载对应的源文件即可在嘉立创 EDA 或 Altium Designer 中打开进行二次开发或生产打样。# -
